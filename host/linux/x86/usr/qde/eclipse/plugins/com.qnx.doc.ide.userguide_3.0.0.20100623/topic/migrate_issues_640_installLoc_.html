IDE location

In earlier versions, by default, the IDE was installed in a different location (in 6.4.1, your default workspace location was home_directory/ide-4.6-workspace on Linux, and C:\ide-4.6-workspace on Windows; in 6.4.0 it was $HOME/QNX640/ide-4.5-workspace; and in 6.3.2 it was $HOME/QNX630/ide-4-workspace.

Now, the IDE is installed as part of the QNX Software Development Platform, and by default, your workspace is located in home_directory/ide-4.7-workspace on Linux, and C:\ide-4.7-workspace on Windows.
