Import wizard

Use the Import wizard to bring files or folders into an existing project from a variety of different sources, such as:


  • an Importing an existing container project into a workspace
  • an Importing an existing project into a workspace
  • another Importing a file system
  • a Importing a QNX Source Package
  • a Importing a QNX mkifs Buildfile
  • a Importing a team project set
  • a Importing an archive file

For details, see Importing projects in the Project and Wizard Properties Reference chapter.
