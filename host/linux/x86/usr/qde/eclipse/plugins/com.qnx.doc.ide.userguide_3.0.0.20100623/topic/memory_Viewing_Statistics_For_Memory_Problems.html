Viewing statistics for memory problems

To view statistics for memory problems (by error type):

  1. From the Memory Problems view, right-click anywhere on the table to open the context menu.
  2. Select Preferencesâ€¦
  3. In the Preferences dialog, select the Expand, Severity, Description and Count columns, and then deselect all of the remaining options.
  4. Click OK.
  5. Right-click, and then select Group By > Type .

The memory problems are grouped by their type, as in example above. The column Count shows the number of problems in the group. The non-aggregate columns without a count show the value of the first problem in the group.
