Configuring output format

To show memory in hexadecimal or ASCII, or as signed or unsigned integers:
  1. In the Memory view's Renderings pane, click the Add Rendering(s) button.

    The Add Memory Rendering dialog appears:

  2. From the dropdown menu, select the memory or expression you wish to add a new memory rendering for, or click Add Newâ€¦ to create a new memory or expression monitor.
  3. Click, Shift-click, or Ctrl-click to choose one or more formats from the Memory Rendering(s) list.
  4. Click OK. Each format you've chosen appears in a separate tab in the Memory view's Renderings pane.
