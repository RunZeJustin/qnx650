Closing projects

If you aren't working on a project, but would like to keep it in the IDE, you can close the project. The benefits of closing a project are that it won't be scanned by the indexer or by SVN tooling and so on, and that closing unused projects can make the IDE more responsive.

To close a project in the IDE:

  1. Select the Project Explorer view.
  2. Right-click your project, and then select Close Project.
