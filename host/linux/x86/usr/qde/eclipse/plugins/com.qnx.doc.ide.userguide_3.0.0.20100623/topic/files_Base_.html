Where Files Are Stored?

These are some of the more important files used by the IDE:

Type of file Default location
Workspace folder $HOME/ide-4.7-workspace
.metadata folder (for personal settings) $HOME/ide-4.7-workspace/.metadata
Error log $HOME/ide-4.7-workspace/.metadata/.log

On Windows, C:/ is used instead of the HOME environment variable or the C:/Documents and Settings/userid directory (so the spaces in the path name don't confuse any of the tools).

Note: You can specify where you want your workspace folder to reside. For details, see the section Running Eclipse in the Tasks chapter of the Workbench User Guide. (To access the guide, open Help > Help Contents , then select Workbench User Guide from the list.)
