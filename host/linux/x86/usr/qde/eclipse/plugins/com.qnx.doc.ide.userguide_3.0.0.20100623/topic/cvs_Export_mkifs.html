Exporting a System Builder project to a Mkxfs build file(s)

To export a System Builder project to a Mkxfs build file(s):

  1. Choose File > Exportâ€¦ The Export wizard appears:

  2. To export your System Builder project to a Mkxfs build file(s), choose QNX > System Builder project to Mkxfs Build File(s) .
  3. Click Next.
  4. Specify a directory in which to create the build file.
  5. When you're done, click Finish.
Note:

To achieve same results as when using IDE build, when you export a build file, ensure that if you're using Override folders, you use -r option to pass the path to the override directory.
