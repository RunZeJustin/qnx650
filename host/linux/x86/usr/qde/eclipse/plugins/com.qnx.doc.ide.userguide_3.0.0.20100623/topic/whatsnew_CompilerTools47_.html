Compiler and Tools


  • The IDE was updated to use gcc version 4.4.; we no longer ship versions 2.95.3 or 3.3.5. For more information about the enhancements, see http://gcc.gnu.org/gcc-4.4/changes.html.
  • The IDE uses GDB 6.7.
  • To be consistent with other platforms, the -n option for the cp command was renamed to be -u. The -n option is still recognized.
  • The low-level atomicity GCC functions were updated so that they no longer use spinlocks, which failed under certain circumstances.
  • The default for Tab completion in ksh is now complete-list, which will list possible choices if the completion doesn't complete. To get the old behavior, add a line with bind ^i=complete to your ~/.kshrc file.
