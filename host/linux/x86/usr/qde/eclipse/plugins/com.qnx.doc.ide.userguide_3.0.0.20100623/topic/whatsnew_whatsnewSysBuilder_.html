System Builder

The System Builder includes the following changes:


  • Enhancements:
    • Replaced the existing terminal with a DSDP terminal view, which includes Com, Telnet and SSH connections.
    • Added the ability to import an entire directory at once.
  • Corrections:
    Note: For additional information about the corrections that were made to the IDE, see List of fixes in the Release Notes.

    • Addressed the permission problem that occurred when starting a TFTP server on Linux & Neutrino.
    • Fixed the export capabilities of the build file export feature.
