Building a project with a custom make target

To build your project with a custom make target:
  1. In the Project Explorer view, right-click a project.
  2. In the context menu, select the Build Make Targetâ€¦ item. The Build Targets dialog appears.
  3. Select your custom target, then click Build.
Note: You can drag files from the Project Explorer to the Make Targets view, or from the Make Targets view to an external editor.
